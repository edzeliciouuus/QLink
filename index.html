QLink Smart Queuing System

A modern, mobile-first queuing system built for schools using PHP, MySQL, and Bootstrap

View on GitHub See Demo

Key Features

Mobile-First Design

Responsive interface that works perfectly on all devices - phones, tablets, and computers.

Role-Based Access

Students, Staff, and Admin roles with appropriate permissions and security.

Real-Time Notifications

SMS and in-app notifications when you're next in line or when your turn is called.

Department Management

Multiple departments with independent queues and specialized management.

Analytics Dashboard

Track wait times, peak hours, completion rates, and system performance.

Security Features

CSRF protection, prepared statements, password hashing, and session security.
